# gumbocr

A straight forward wrapper for [Gumbo](https://github.com/google/gumbo-parser).

## Installation

Add it to `Projectfile`

```crystal
deps do
  github "zamith/gumbocr"
end
```

## Usage

```crystal
require "gumbocr"
```
